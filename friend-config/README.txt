CS2 DEATH MUTER - SETUP
=======================

Get muted for 3 seconds when you die in CS2.


STEP 1: Run the installer
-------------------------
Double-click "install-gsi.bat"


STEP 2: Restart CS2
-------------------
If CS2 is running, restart it.


STEP 3: Register in Discord
---------------------------
Type in the Discord server:

    /register YOUR_STEAM_ID

To find your Steam ID:
1. Go to https://steamid.io
2. Paste your Steam profile URL
3. Copy the "steamID64" (starts with 7656...)

Example: /register 76561198012345678


Done. Join voice and play.


TROUBLESHOOTING
---------------
- Not working? Restart CS2.
- Wrong Steam ID? /unregister then /register again.
- Still broken? Check bot host has server running.
